这个程序主要介绍了在网页上显示飞行文字。
用户可以看到网页上的文字飞一样，越来越大，越来越近。
通过这个例子读者可以学习到如何在网页上展示飞行文字。


// 如果字体尺寸过大
if (fontsize > getSize().height){
fontsize = 0;}
这段代码的作用是当显示字体的尺寸大于applet的尺寸时，就重新设置显示字体的大小为0.
这样就可以使文字有不断飞行的效果。

update（）方法中
// 绘制Image对象
g.drawImage(image, 0, 0, this);
//增大字体大小
fontsize++;
