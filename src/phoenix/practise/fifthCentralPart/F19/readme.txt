该程序介绍了程序启动界面是如何设计的，当你看完这个程序启动界面的代码之后，
你会发现原来程序启动界面并非你想像中的困难，它只需几行代码就可以实现一个漂亮的启动界面。

MyWindow()构造函数：实例化变量，构造窗口，设置光标状态。
start()函数：设置窗口前端显示，实例化线程，运行线程。
run()函数： 线程运行之后会自动运行，它的功能包括：显示窗口、控制进度条、释放窗口、显示这窗口。
showFrame()函数：进度条完成之后要显示的主窗口在这个函数中执行。

progress = new JProgressBar(1, 100); 该程序实例化进度条，把进度条数分为100份。

setLocation((screen.width - getSize().width) / 2,(screen.height - getSize().height) / 2)；
该程序设置窗口的位置，在显示器的中间位置显示出来。
